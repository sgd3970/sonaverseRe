# 로고 이미지 디렉토리

이 폴더에 다음 로고 이미지 파일들을 넣어주세요:

## 필요한 파일 목록

1. **한국어 로고** - `ko_logo.png` 또는 `ko_logo.svg`
   - 한국어 텍스트가 포함된 회사 로고
   - 권장 크기: 가로 120-160px (2배 해상도: 240-320px)

2. **영문 로고** - `en_logo.png` 또는 `en_logo.svg`
   - 영문 텍스트가 포함된 회사 로고
   - 권장 크기: 가로 120-160px (2배 해상도: 240-320px)

3. **심볼 로고** - `symbol_logo.png` 또는 `symbol_logo.svg`
   - 텍스트 없이 회사 심볼만 있는 로고
   - 권장 크기: 40x40px (2배 해상도: 80x80px)
   - 용도: 파비콘, 모바일 메뉴 등

## 파일 형식 권장사항

- **SVG 형식 (권장)**: 확대/축소 시 선명도 유지, 파일 크기 작음
- **PNG 형식**: 투명 배경 지원, 2배 해상도(@2x) 준비

## 사용 위치

- **Header**: 언어에 따라 한국어/영문 로고 자동 전환
- **Footer**: 언어에 따라 한국어/영문 로고 자동 전환
- **모바일**: 공간이 제한될 경우 심볼 로고 사용 가능

## 예시 구조

```
public/logo/
├── ko_logo.svg          # 한국어 로고 (SVG)
├── ko_logo@2x.png      # 한국어 로고 (PNG 2배 해상도)
├── en_logo.svg          # 영문 로고 (SVG)
├── en_logo@2x.png      # 영문 로고 (PNG 2배 해상도)
├── symbol_logo.svg      # 심볼 로고 (SVG)
└── symbol_logo@2x.png  # 심볼 로고 (PNG 2배 해상도)
```

## 최적화 팁

1. SVG 파일은 SVGO로 최적화
2. PNG 파일은 TinyPNG 등으로 압축
3. 로고 배경은 투명으로 설정
4. 다크모드 대응이 필요한 경우 별도 버전 준비
